# Facilities Halls Booking System

A web-based platform for managing campus facilities, hall bookings, tickets, and maintenance requests in one place.

## Features
- Hall and room booking management
- Ticket submission and tracking
- Maintenance request handling
- Admin dashboard for approvals and updates
- User authentication and role-based access
- Notifications for booking and ticket status changes

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
- Styling: CSS / Tailwind
- Build Tools: Maven, npm
## Getting Started

### Prerequisites
- Node.js
- Java 17+ or compatible version
- Maven
- MySQL
